# CLI

## Installation

```
npm install -g @serieseight/cli
```

and then:

```
s8 setup
```

## Usage

```
s8 <command>
```

To view all available commands run `s8 --help`.

## Development

Clone the repository from https://github.com/serieseight/cli.

Install dependencies with `npm install` (avoid yarn as internals use npm).
